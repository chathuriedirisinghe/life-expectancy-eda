# 🌍 Life Expectancy Data Analysis

## Overview
This project performs exploratory data analysis (EDA) on global health data to understand factors influencing life expectancy.

## Dataset
WHO Life Expectancy dataset (2938 rows, 22 variables)

## Key Findings
- Most countries have life expectancy between 60–80 years
- Higher GDP is associated with longer life expectancy
- Mortality rates (adult, infant) strongly reduce life expectancy
- Education and income positively influence life expectancy

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
